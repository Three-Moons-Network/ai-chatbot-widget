Configuration

To embed the widget in your own website, add this code to your HTML:

<div id="chat-widget"></div>
<script>
  window.ChatWidgetConfig = {
    apiEndpoint: "https://your-api-id.execute-api.region.amazonaws.com",
    sessionId: "user-" + Math.random().toString(36).substr(2, 9)
  };
</script>
<script src="https://your-domain.com/widget/widget.js"></script>
Note: Replace the apiEndpoint with your actual Lambda API Gateway endpoint from the Terraform deployment.

Configuration Options

The ChatWidgetConfig object accepts the following options:

Required:
• apiEndpoint - Your Lambda API Gateway endpoint
• sessionId - Unique identifier for this user/conversation

Optional:
• title - Chat window title (default: "Chat Support")
• greeting - Initial greeting message
• position - Widget position: bottom-right, bottom-left, top-right, top-left
• width - Widget width in pixels (default: 380)
• height - Widget height in pixels (default: 500)

API Endpoints

POST /chat

Send a message and receive an AI response.

{
  "session_id": "user-123",
  "message": "How do I reset my password?"
}

GET /chat/{session_id}

Retrieve the full conversation history for a session.

GET /chat/user-123

Getting Started

  1. Clone the repository from GitHub
  2. Configure your Anthropic API key in terraform.tfvars
  3. Run ./scripts/deploy.sh to build the Lambda package
  4. Run terraform apply to deploy to AWS (see README.md for details)
  5. Update the apiEndpoint in your configuration
  6. Embed the widget script in your website
